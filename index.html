Rangkuman Modul HTML

Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur dan konten halaman web. HTML memberikan kerangka dasar untuk menampilkan teks, gambar, tautan, dan elemen lainnya di browser.

Struktur Dasar HTML

Setiap dokumen HTML dimulai dengan deklarasi tipe dokumen, diikuti oleh elemen dasar. Berikut adalah struktur sederhana:

<!DOCTYPE html>
<html>
<head>
<title>Judul Halaman</title>
</head>
<body>
<h1>Selamat Datang di Website Saya</h1>
<p>Ini adalah paragraf pertama saya.</p>
</body>
</html>

Pengenalan 10 Tag HTML Dasar

  1. <html>: Elemen utama yang menandai awal dan akhir dokumen HTML.
  2. <head>: Bagian yang berisi informasi tentang dokumen, seperti judul dan meta data.
  3. <title>: Menentukan judul halaman yang muncul di tab browser.
  4. <body>: Bagian utama dari halaman yang berisi konten yang terlihat oleh pengguna.
  5. <h1> hingga <h6>: Tag untuk judul, dengan <h1> sebagai yang paling penting dan <h6> sebagai yang paling sedikit penting.
  6. <p>: Digunakan untuk membuat paragraf.
  7. <a>: Tag untuk membuat tautan (hyperlink). Contoh: <a href="https://www.example.com">Kunjungi Example</a>.
  8. <img>: Untuk menyisipkan gambar. Contoh: <img src="gambar.jpg" alt="Deskripsi Gambar">.
  9. <ul> dan <li>: Untuk membuat daftar tidak terurut. Contoh :
    <ul>
    <li>Item pertama</li>
    <li>Item kedua</li>
    </ul>
  10. <div>: Elemen blok yang digunakan untuk mengelompokkan konten dan mengatur layout.
